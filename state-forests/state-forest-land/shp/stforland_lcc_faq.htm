What does this data set describe?
    Title: stforland_lcc
    Abstract:
    This dataset contains 1:24000-scale State Forest property boundary polygons for the State of Virginia. In this case, State Forests are defined as forest properties owned and/or managed by the Virginia Department of Forestry. For example, two of the properties in this dataset are owned by the Virginia Department of Conservation and Recreation, but are managed by the Virginia Department of Forestry and so are included. Property boundaries were derived using several methods and varying bases data sources.
    Supplemental_Information:
    The property boundary for the Chilton Woods State Forest was derived from a hard-copy map of the surveyed property lines and was digitized on-screen using both USGS 7.5' DRGs and USGS DOQQ data. The Devil's Backbone property boundary was derived from a hard-copy delineation on a USGS 7.5' topo map, and was digitized on-screen over the corresponding USGS 7.5' DRG data.

    Other property boundaries were delineated on uncontrolled aerial photography which were transferred to USGS Topo maps. The hard-copy maps were then digitized using a digitizing table. There are plans to overlay these property boundaries onto DOQQ images and to correct any known discrepancies.
