Description:
Abstract:
This dataset contains forest stand delineations for the Appomattox-Buckingham State Forest. Stand boundaries were originally digitized at a scale of 1:24000 from hardcopy USGS 7.5' Quadrangles. The stand boundaries were updated February 2001 using 1994 & 1996 USGS DOQQ imagery to a scale of 1:12000. Much of the information used to attribute this dataset comes from data historically collected and managed by the State Forests.
Purpose:
These data are used to inventory, track, and manage forest lands on the Appomattox-Buckingham State Forest.
Supplemental_Information:
Stand information has historically been managed and maintained using a Forest Stand Card Filing System where stand data were recorded on cards. This method was updated by delineating stand boundaries on hardcopy 1:24k USGS 7.5' quadrangle map and converting these delineations into digital formats. Boundaries were updated using map-grade GPS data as needed. Attributes were also updated intermittently, as needed. From December 2000-February 2001 DOF updated stand boundaries using 1994 and 1996 1:12000 USGS DOQQ imagery, as well as some GPS data.

Data_Quality_Information:
Completeness_Report:
The stand delineations are based on 1994 and 1996 imagery and, as such, will reflect nuances of this imagery. Stand boundary updates made with map-grade GPS are more recent and more accuracte, but are a small percentage of the entire dataset. Attribute data are believed to be correct; however final update checks have yet to be performed. The stand age field is subject to annual update.
